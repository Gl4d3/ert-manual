4.4 Network Meter Management

This window enables the user to navigate the entire network tree, view the meters installed in different network elements and the readings for these meters. Figure 4.15 shows how the window looks like in ERT.

Figure 4.18: Network Meter Management Window

The Feeder Management window can be divided into 4 sections. These are:

  1. Location search
  2. Network Tree/Structure
  3. Secondary Substations
  4. List of Meters

Figure 4.16 shows the sections numbered respectively

Figure 4.16: Network Meter Management Sections

1. Location Search

This sections allows the user to filter the results of the network tree based on the region and county they belong to.

2. Network Tree/Structure

This section enables the user to browse the energy network from the transmission substations to the primary feeders. The network elements present in the network tree are in the following order:

  • Transmission Substation
  • Transmission Feeder
  • Primary Substation
  • Primary Feeder

All transmission substations are under one parent, which is named ‘Kenya Power’
