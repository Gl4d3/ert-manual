Introduction

The Energy Reporting Tool (ERT) is a web-based application designed to enable energy companies to calculate energy losses over time. The tool generates detailed reports to help identify the causes of energy losses within an energy network.

Accessing the System

Navigating the System

Upon successful login, users will be directed to the Home window. The top bar includes the application title, a feedback button, a help button, and a user profile button. The sidebar provides navigation to various sections of the application, including:

  • Home
  • Vertical Balances
  • Horizontal Balances
  • Network Meter Management
  • Meter Readings
  • Exceptional Report
  • System Users
  • System Parameters
  • Manage Profiles

ERT Windows

Each window in the ERT system is designed to facilitate specific tasks. These include:

Home

The Home window displays a summary of reports, including general performance reports, feeder distributions, and more.

Vertical Balances

This window allows users to generate vertical energy loss reports, which track energy loss from transmission to customer consumption. Users can select various criteria such as calculation type, date period, and network options to generate reports.

Horizontal Balances

The Horizontal Balances window enables the generation of reports on energy loss across administrative boundaries. Users select the financial year to generate reports that track these losses.

Network Meter Management

This section allows users to manage and monitor network meters, including location searches, network structures, and meter lists.

Exceptional Reports

This window is used to generate reports on exceptional cases, such as meters on feeders that are not in the database, large power customers, and elements with unknown locations.

User Management

Manage system users, their profiles, and roles. Users can create, edit, or delete profiles and set system parameters in this section.

Generating Reports

ERT allows users to generate various types of reports, including:

Report Type Description
Monthly Energy Balances Tracks monthly energy losses for each feeder meter.
Cumulative Energy Balances Aggregates energy losses over a user-defined period, useful for annual, semi-annual, or quarterly reports.
Per Region Report and per County Summarizes cumulative energy balances by region and county.
Execution Analysis Report Tracks the execution status of different report generations over time.
Anomalies Reports Master report that contains anomalies and areas of concern within both the system and energy balance.
